Upload PDFs, presentations, and documents. System runs secure file signature checks dynamically to protect curation pipeline integrity.

Semantic Curriculum Mapping

AI-Driven Synthesis

CHAPTER II

Curation Pipeline Architecture

01

Ingestion & OCR Extraction

Sources are chunked dynamically, converting text, math equations, and tables into searchable markdown vectors.

02

Fact Mining

LLM agents mine properties, benefits, and misconceptions, logging trace parameters dynamically for observability.

03

Conflict Resolution

Synthesizes overlapping source material into consolidated facts based on source-trust ratings and resolution heuristics.

04

Textbook Compiling

Assembles finalized definitions, key FAQs, LaTeX variables, and markdown images into a structured curriculum page.
